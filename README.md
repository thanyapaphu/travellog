# Travel Log (mobile-first)

Small mobile-friendly travel log web app:

- Home → button **“Tap into this trip”**
- Name page → typing updates the **logbook illustration live**
- Q2 `[2Tokyo-Yoroniku]` → choose place (dropdown), “How did you feel?” (text), “Would you go back again?” (one option)
- Q3 `[3Karuizawa-Sanu]` → same
- Q4 `[4Hakuba-Tsugaike]` → same
- Answers page → shows the exact `answerX-...` key based on your choice mappings
- End page → start over

## Run locally

```bash
cd travel-log
npm install
npm run dev
```

## Build

```bash
cd travel-log
npm run build
```
